# 🎓 IA Academy - Sales Page

Página de vendas de alta conversão para o curso IA Academy. Focada em infoprodutos e educação sobre Inteligência Artificial.

## 🛠️ Tecnologias
* **Estrutura**: HTML Semântico + CSS Customizado
* **Performance**: Otimizada para carregamento rápido (SEO)
* **Marketing**: Integração com Meta Pixel e botões de checkout (Cakto).

## 🌟 Funcionalidades
* **Design Persuasivo**: Focado em conversão de leads.
* **Prova Social**: Seção de depoimentos e números de alunos ativos.
* **Countdown**: Gatilhos de escassez integrados.
